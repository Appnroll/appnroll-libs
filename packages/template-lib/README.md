<h1 align="center">
  Template lib
</h1>

Template lib catalog to share how to setup package for the next lib in the monorepo.

## Install

```
npm install @appnroll/template-lib

# or

yarn add @appnroll/template-lib
```
## Usage
```js
const fn = require("@appnroll/template-lib")
fn(["A", "A", "B"]) // -> ["A", "B"]
```
